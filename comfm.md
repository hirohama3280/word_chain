# 各自作業の残し方

### HTMLのコメントフォーマット
◎開始時
```html
<!-- 
==================
*No：
*担当者：
*日付：
*変更コメント：
==================
-->
```
◎終了時
```html
<!-- 
==================
担当者：
No.〇END
==================
-->
```
---

### CSSのコメントフォーマット
◎開始時
```css

==================
*/
```
◎終了時

```css
==================
*/
```
